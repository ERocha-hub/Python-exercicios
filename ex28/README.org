* Exercício 28 - Jogo adivinhação v1.0
Este programa ramdomiza um número de 0 a 5 e pede um input para o usuário tentar adivinhar. Responde se acertou ou não.

Curiosidade: =\033= é um "escape code" do ANSI para dar uma saída de código ao terminal. Neste programa, usamos para representar as cores do terminal na saída do código. São equivalentes:
- Octal: \033
- Hexadecimal: \x1B
- Unicode: \u001b
- Tecla Ctrl: ^[

Esses códigos são criados assim:
- Começam com ~\x1B[~ (qualquer código especificado + [)
- Então são adicionados um número arbitrário de inteiros separados por ~;~
- Terminam em ~m~
